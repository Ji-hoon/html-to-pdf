자기소개

▸ 실증적인 경험을 바탕으로 빠른 성장 지향

ERP 제품의 UI 디자인 작업과 스토리북을 활용한 디자인 시스템 구축을 모두 담당하여 디자인 리뉴얼 개발을 빠르게 주도한 경험이 있습니다. 부트캠프에서의 팀·개인 프로젝트를 통해 백엔드와 프론트엔드 역할을 모두 경험해 개발 프로세스 전반에 대한 이해도를 높였습니다.

▸ SaaS 분야의 넓고 깊은 도메인 지식 보유

복잡한 Workflow를 이해하기 쉽게 설계하고 기획·디자인하면서 얻은 인사이트를 바탕으로, 정해진 산출물 없이도 요구사항 기반으로 빠르게 스펙을 도출해 화면 설계와 개발을 진행할 수 있습니다.

▸ 요구사항 분석·일정 관리·문서화에 강점

복잡한 비즈니스 로직을 빠르게 분석하고 태스크 단위로 나눠 스프린트 미팅 없이도 개발 일정을 준수합니다. 아이데이션 문서, 포스트맨 API 명세, 소셜 로그인 플로우차트 등 문서화·도식화 작업을 주도한 경험이 있습니다.
